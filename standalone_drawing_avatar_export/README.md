# お絵描きアバター単体版

`standalone-drawing-avatar.html` をブラウザで開くと、元アプリの「お絵描きで新キャラ作成」部分だけを単体で動かせます。
ブラウザのセキュリティ設定で `file://` 直開き時にスクリプトやCSSがブロックされる場合は、任意のローカルHTTPサーバー配下で開いてください。

## ファイル

- `standalone-drawing-avatar.html` : 起動用HTML
- `standalone-drawing-avatar.css` : 画面デザイン
- `standalone-drawing-avatar.js` : 描画・レイヤー・PNG出力処理

## 主な機能

- 顔ベース、目、口、前髪、後ろ髪、アイテムレイヤーへの描画
- 後ろ髪を前髪の下に表示する合成順
- ブラシ、塗りつぶし、消しゴム
- ブラシ太さと柔らかさ
- 手ぶれ補正（線を滑らかに）
- ペンタブレットの筆圧で太さが変化（筆圧チェックON時）
- ズーム＆パン
  - ホイールでズーム、Space または中ボタンドラッグで画面移動
  - ツールバーの −／＋／全体 ボタンでも操作可能
- ブラシサイズのカーソルプレビュー（円表示）
- 6表情のライブプレビュー（右パネル上部）
  - クリックするとキャンバスの目・口がその組み合わせに切り替わる
- レイヤー一覧のグループ表示（ベース／目／口／髪／アイテム）
- 文字説明なしの色スウォッチ
  - 色入力で選択中スウォッチ自体の色が変わります
- 各レイヤーへのPNG読み込み
- 読み込んだPNGの位置・大きさ調整
- 完成PNG生成後もそのまま描き直し可能
- プロジェクトJSONの保存・読み込み
- 元に戻す履歴は30回まで（描画キャンバスと読み込みPNGの状態が対象です。レイヤー選択や表示切替などのUI状態は対象外です）

## キーボードショートカット

- `B` ブラシ / `G` 塗りつぶし / `E` 消しゴム
- `[` `]` ブラシ太さ
- `Ctrl+Z` 元に戻す / `Ctrl+Y` やり直し
- `+` `-` ズーム / `0` 全体表示 / `Space+ドラッグ` 画面移動

## 書き出し

「完成PNGを生成」を押すと、以下を生成します。

- `eyes-open-mouth-closed.png`
- `eyes-open-mouth-half.png`
- `eyes-open-mouth-open.png`
- `eyes-closed-mouth-closed.png`
- `eyes-closed-mouth-half.png`
- `eyes-closed-mouth-open.png`
- `front-hair.png`
- `back-hair.png`
- `item-*.png`

## 注意

この単体版は他PCで改造しやすいように、元アプリから独立したHTML/CSS/JSだけで構成しています。
元アプリ本体への保存連携や.purupuru保存処理は含めていません。
